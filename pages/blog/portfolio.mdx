## Stack

Tools I used to make this site

*   [webpack](https://webpack.js.org/) for bundling and as a dev server
*   [TypeScript](https://www.typescriptlang.org/) for input-output checks
*   [React](https://reactjs.org/) for UI-data binding
*   [MDX](https://mdxjs.com/) to render markdown, like this page
*   HTML written with accessibility in mind
*   [Next.js](https://nextjs.org/) for routing and server-side rendering
*   [Cognito](https://aws.amazon.com/cognito/) for authentication
*   [S3](https://aws.amazon.com/s3/) and [CloudFront](https://aws.amazon.com/cloudfront/) for static asset hosting
*   [DynamoDB](https://aws.amazon.com/dynamodb/) for data storage
*   [Lambda](https://aws.amazon.com/lambda/) and [API Gateway](https://aws.amazon.com/api-gateway/) for interacting with the database
*   [Route53](https://aws.amazon.com/route53/) for DNS
*   [Amplify](https://aws.amazon.com/amplify/) for continuous deployments and cross-device data sync
